# My Dotfiles

This repository contains my personal configuration files (dotfiles) for various tools and applications.

## Requirements

### Git

Linux & Windows (WSL)

```bash
sudo apt install git
```

MacOS

```bash
brew install git
```

### Stow

Linux & Windows (WSL)

```bash
sudo apt install stow
```

MacOS

```bash
brew install stow
```

## Installation

```bash
git clone https://github.com/Aquabbas/dotfiles.git ~/dotfiles
cd ~/dotfiles
stow --target .
```
